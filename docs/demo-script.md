# Demo Script

## Flow
1. Open home page and show architecture navigation.
2. Navigate to Campaign Lab and run Vue lead scoring.
3. Navigate to Lead Workbench and run Blazor lead scoring.
4. Submit a lead through API endpoint and show generated reference.
5. Call health endpoint and explain deployment readiness.

## API checks
- POST /api/leads/score
- POST /api/leads/submit
- GET /health/ready
